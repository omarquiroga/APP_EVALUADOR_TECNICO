# Terminologia UI

Glosario visible aprobado para la interfaz operativa:

- assessment -> evaluacion financiera
- input -> insumo financiero
- submitted -> radicado
- current -> vigente
- history -> historial
- details -> detalle
- process -> proceso
- bidder -> proponente
- dossier -> expediente
- validation -> validacion
- rejection cause -> causal de rechazo
- consolidation -> consolidacion

Reglas de uso:

- Mantener los textos visibles en espanol operativo.
- No renombrar modelos, rutas o variables internas solo por traduccion.
- Evitar anglicismos innecesarios en pantallas, mensajes y breadcrumbs.
